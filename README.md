# Pauschal

A variable sans-serif font inspired by [Atkinson Hyperlegible](https://github.com/googlefonts/atkinson-hyperlegible)

![pauschale Satzschrift](images/title-card.svg)

- ASCII only (for now)
- font weights 200 - 600
